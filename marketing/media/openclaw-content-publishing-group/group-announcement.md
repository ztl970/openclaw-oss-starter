# OpenClaw内容与发布群公告

本群已更名为：

`OpenClaw内容与发布群`

## 群定位

本群用于承接已经过中枢整理后的对外内容工作，主要负责：

1. LinkedIn 内容整理与发布
2. 小红书内容改写与发布准备
3. 发布前文案协同
4. 发布后复盘与下一条内容规划

## 分工

- `OpenClaw中枢群`
  - 负责内部整理
  - 负责 public-safe 判断
  - 负责决定哪些内容能进入 GitHub / ClawHub 的公开链路

- `OpenClaw内容与发布群`
  - 负责对外内容运营
  - 负责平台改写
  - 负责发布执行协同

- 果妈
  - 负责 LinkedIn 发布

- 果爸
  - 负责小红书发布

## 一句话规则

中枢群负责“能不能公开”，本群负责“怎么发出去”。
