# 不应公开内容清单

## 目的

这一页用于明确哪些内容默认不进入公开链路。

公开链路包括：

- GitHub
- ClawHub
- 飞书公开项目知识库
- 对外发布文案

## 绝对不应公开的内容

### 1. 真实身份信息

- 真实姓名
- 身份证类信息
- 手机号
- 邮箱
- 精确住址

### 2. 家庭与未成年人可识别信息

- 孩子真实姓名
- 学校名称
- 班级信息
- 精确课程安排
- 可追踪到未成年人的独特信息

### 3. 账号与系统标识

- 群 ID
- 账号 ID
- chat ID
- token
- session code
- secret / key / password

### 4. 私密运行态内容

- 私有记忆
- heartbeat
- 内部状态记录
- 私人执行日志
- 只供本地实例理解的运行态上下文

### 5. 私人家庭规则

- 只适用于你家当前成员的细则
- 不能抽象复用的家庭制度
- 带具体家庭作息和身份绑定的安排

### 6. 私有业务或敏感协作信息

- 未公开业务线索
- 私有客户信息
- 内部合作细节
- 私有联系人

## 可以保留但必须先抽象的内容

- 个人场景中提炼出的流程
- 家庭场景中提炼出的通用模板
- 本地工作流中提炼出的通用规则

抽象要求：

- 改成通用角色
- 移除身份线索
- 移除账号线索
- 移除精确私人时间与地点

## 一句话规则

凡是会暴露“这个内容只属于你本人、你家庭、你的账号或你的私有运行态”的，都不应公开。

## 发布前检查关系

如果某项内容命中本页，应直接判定：

- 不进入公开链路
- 只保留本地
- 必要时先做抽象化改写
