# 47_Main 极简短口令

## 适用前提

这些口令默认只适用于正式中枢群。

因为正式中枢群已经默认继承：

- 当前项目
- 当前仓库
- 当前协作链

## 推荐极简口令

### 1. 看今天哪些值得更新 GitHub

```text
请从 main 看今天哪些值得更新 GitHub。
```

### 2. 判断要不要同步 ClawHub

```text
请从 main 判断这次要不要同步 ClawHub。
```

### 3. 判断要不要同步飞书知识库

```text
请从 main 判断这次要不要同步知识库。
```

### 4. 做一次日常同步检查

```text
请从 main 做一次日常检查。
```

### 5. 做一次周度复盘

```text
请从 main 做一次周复盘。
```

### 6. 处理一次 GitHub 项目工作

```text
请从 main 处理这次 GitHub 工作。
```

### 7. 做一次多项目日常检查

```text
请从 main 做一次多项目日常检查。
```

## 使用原则

- 在正式中枢群优先用极简口令
- 默认不重复解释 repo、路径、执行角色
- 只有要切换项目时，才额外补上下文
