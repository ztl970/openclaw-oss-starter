# Main Daily Short Commands

Use these short commands in the Feishu `main` chat window for routine work.

## 1. 日常同步检查

```text
请从 main 做一次日常同步检查。
```

含义：

- 默认检查 openclaw-oss-starter
- 看 GitHub、ClawHub、飞书知识库是否需要同步

## 2. 知识库同步

```text
请从 main 同步 OpenClaw 开源项目知识库。
```

含义：

- 默认同步 `11_OpenClaw开源项目` 目录下标准页面
- 由 `media` 执行

## 3. GitHub 项目处理

```text
请从 main 处理这次 GitHub 项目工作。
```

含义：

- 默认按 `openclaw-oss-starter` 处理
- 先判断任务类型，再决定是否需要公开同步

## 4. 新 skill 上线流程

```text
请从 main 处理一次新 skill 上线流程。
```

含义：

- 先做本地结构检查和仓库更新
- 再决定是否同步 GitHub、ClawHub、飞书知识库

## 5. 周度复盘

```text
请从 main 做一次本周项目复盘。
```

含义：

- 检查本周更新
- 检查双 hub 是否一致
- 整理下一步待办

## 6. 今日哪些值得更新到 GitHub

```text
请从 main 回看今天 openclaw-oss-starter 的工作，判断哪些值得更新到 GitHub。
```

含义：

- 只看默认项目今天的实际工作结果
- 不看泛化聊天，不看抽象“自己的一天”
- 先判断公开价值，再判断同步范围
- 必须回报 `同步范围判断`

## 使用建议

- 日常对话优先用短指令
- 如有特殊要求，再在下一句补限制条件
- 若未指定其他仓库，默认使用 `openclaw-oss-starter`
