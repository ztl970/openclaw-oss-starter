# Minimal Example

This folder shows one copyable local workflow pattern for `openclaw-oss-starter`.

## Example workflow

1. Run the repository validation helper.
2. Check the quota-aware switching script.
3. Review the routing and boundary docs before changing any local instance rules.

## Suggested commands

```bash
./validate_repo.sh
./cron/codex_quota_autoswitch.sh
bash ./examples/pack_and_validate.sh
```

## What to copy

- the public-safe repo layout
- the validation habit
- the routing/boundary doc pattern
