# Customization Guide

This skill can be adapted for different Mac multi-instance layouts.

## Customizable fields

- instance labels
- workspace roots
- boundary wording
- pack inclusion rules
- routing/index wording

## Suggested adaptation patterns

- main / workbench / publish / archive / private
- main / work / review / release / archive
- main / local / staging / publish / backup

## Public-safe reminder

Before publishing a customized variant, remove:

- real names
- household-specific roles
- private IDs or account data
- sensitive schedules
- secrets and local-only runtime files
