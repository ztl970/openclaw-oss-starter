# Customization Guide

This skill can be adapted for different types of structured practice.

## Customizable fields

- task label
- reminder wording
- completion check wording
- follow-up wording
- sender role label

## Suggested adaptation types

- dance practice
- instrument practice
- sports drills
- reading practice
- speaking practice

## Public-safe reminder

Before publishing a customized variant, remove:

- real names
- real institutions
- real schedules tied to identity
- private IDs
- sensitive personal details
